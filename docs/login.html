Get Going

Login to Sunshower.io

Once you log in, you'll be prompted to create a system, so you'll need some read-only, programmatic access IAM credentials for us to use. Then we'll discover your ASGs and EC2 instances all around the world and help you optimize them!

Please note that we can't crunch numbers that we don't have. We currently rely on CloudWatch to get information about your system. If you see that an instance has next to a metric (typically memory) for an instance, this means that you don't have the CloudWatch agent on the instance and will need to add it.

We hope you'll find it useful! Please let us know via email or on social media. We'd love to hear about what other features or clouds you might be interested in. And if you know someone else who could use us, please pass us along!

Sunshower.io is a cloud management platform that empowers organizations to manage the entire lifecycle of their cloud infrastructure, regardless of financial, technical, and personnel resources.
